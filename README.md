<br><br>

## 기간
- **2024.04~** 






<br><br>

## 프로젝트 목표
- 객체 지향 설계를 토대로 개발하기
- 지속적으로 성능 개선 및 리팩토링하기
- 티켓팅 과정에서 동시성 이슈를 고려하여 구현하기
- 가독성, 유지보수성를 향상시키기 위해 클린코드를 지향합니다.
